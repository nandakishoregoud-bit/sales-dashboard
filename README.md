# 📊 Sales Analytics Dashboard with AI Insights

## 🚀 Project Overview

This project is an end-to-end **data analytics dashboard** built using Python. It analyzes sales data to generate business insights and provides an interactive web interface for users to explore the data.

The dashboard includes:

* Data cleaning and preprocessing
* Sales and profit analysis
* Interactive filters
* KPI metrics
* AI-based insights
* Cloud deployment

---

## 🎯 Objectives

* Analyze sales performance across categories, regions, and cities
* Identify high-performing and low-performing areas
* Build an interactive dashboard for business decision-making
* Integrate AI-based insights for automated analysis

---

## 🛠️ Tech Stack

* Python
* Pandas (data analysis)
* Matplotlib (visualization)
* Streamlit (web dashboard)
* GitHub (version control)
* Streamlit Cloud (deployment)

---

## 📂 Dataset

* Dataset: Sample Superstore Sales Data
* Records: ~9994 rows
* Columns include:

  * Category, Sub-Category
  * Sales, Profit, Quantity
  * Region, City, State

---

## 🧹 Data Cleaning

Performed the following steps:

* Checked for missing values → none found
* Removed duplicate records (17 duplicates removed)
* Verified data types

---

## 📊 Data Analysis

### Key Analysis Performed:

* Sales by Category
* Profit by Category
* Sales by Region
* Top 5 Cities by Sales

### Key Findings:

* **Technology** is the most profitable category
* **Furniture** has high sales but very low profit
* **West region** generates highest sales
* **New York City** is the top-performing city
